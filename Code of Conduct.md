
**Project:** Interactive Drug Identity & Comparison Dashboard

## 1. Purpose

This project aims to provide a transparent and respectful environment for collaboration. We are committed to creating a professional, inclusive, and academically responsible workspace for all contributors.

This Code of Conduct outlines expected behavior for team members and contributors.

---
## 2. Our Commitment

We commit to:
- Respectful communication
- Equal participation
- Academic integrity
- Professional collaboration
- Inclusive and discrimination-free conduct

All contributors are expected to uphold these standards.

---

## 3. Expected Behavior

All participants agree to:
- Communicate respectfully and professionally
- Provide constructive feedback
- Listen actively to team members
- Accept differing technical opinions
- Credit contributors appropriately
- Use proper citation for external resources
- Follow GitHub workflow standards (branches, pull requests, reviews)

We value collaboration over competition.

---

## 4. Unacceptable Behavior

The following behaviors will not be tolerated:
- Harassment, discrimination, or personal attacks
- Dismissive or hostile communication
- Plagiarism or academic dishonesty
- Copying code without proper attribution
- Repeated failure to complete agreed responsibilities without communication
- Unauthorized modification of othersâ€™ contributions

Any violation will be addressed immediately.

---

## 5. Academic Integrity

This project is conducted within an academic setting. Therefore:
- All work must comply with course integrity policies
- External libraries and APIs must be properly cited
- AI tools (if used) must comply with course guidelines
- No unauthorized collaboration outside approved team members

Integrity is non-negotiable.

---

## 6. Communication Standards

We agree to:
- Respond to team communication within 24 hours
- Inform the team in advance if unavailable
- Discuss disagreements calmly and logically
- Use GitHub Issues for task clarity

Professional tone is expected in all written and verbal communication.

---

## 7. Conflict Resolution

If conflicts arise:
1. Discuss the issue respectfully within the team
2. Attempt resolution through open communication
3. If unresolved, consult the course instructor or TA

We prioritize solutions over blame.

---

## 8. Enforcement

Violations of this Code of Conduct may result in:
- Task reassignment
- Formal documentation of the issue
- Escalation to course staff (if required)

This policy exists to protect the integrity of the team and the project.

---

## 9. Acknowledgment

By contributing to this repository, all members agree to abide by this Code of Conduct.
